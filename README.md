#Primo progetto corso IoT (gruppo Argenti Enrico ed Eupani Alessio)

## Indroduzione al progetto

Si sono implementati quattro fra i più diffusi ed affermati algoritmi per i sistemi IoT
applicandoli ad un progetto consistente nel mettere in comunicazione una flotta di droni 
ad uso commerciale con un backend in cloud. 

Gli algoritmi sfruttati sono: 

* HTTP
* MQTT
* CoAP
* AMQP

Ad ogni branch corrisponde l'implementazione di uno di essi.
